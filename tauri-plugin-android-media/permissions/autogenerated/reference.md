## Default Permission

Default permissions for the android-media plugin. Allows querying MediaStore audio metadata and computing audio hashes via ContentResolver.

#### This default permission set includes the following:

- `allow-has-audio-permission`
- `allow-request-audio-permission`
- `allow-query-audio-metadata`
- `allow-audio-hash`
- `allow-register-listener`
- `allow-remove-listener`
